  2. Qus:Define version control and give one example.

    Ans:Allows you to record changes to files and keep track of different versions of software. An example is Github where you can save your files or project using a program within a program. It also allows other coders and programmers to view each other's work.

  4. Qus:What tag is used to create a paragraph in html? Give an example.

  5. Qus:Identify the odd one from these tags: <span> <p> <a> <i> and state the reason?

    Ans:<Span> is the odd one out as it is an element you can use to style in CSS whereas the rest are tags.

  7. Qus:What element is used to create images? Name its attributes.

    Ans:The element used to create images is <img>. Attributes of the image tag are <img src="url"> the use of alt="cat on a chair" and width, height and style.

  9. Qus:Give an example of a colour picker input.
